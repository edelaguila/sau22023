Capacitación - Asignacion Grupos


Asignación de Grupos

Tiene como objetivo asignar un encargado a un grupo.




Formulario

En la siguiente figura podemos observar los campos por llenar en el formulario de asignación de grupo.




Llenado de datos

Para agregar datos se deben de llenar los campos, al llegar a un campo deshabilitado, presionar el boton con el signo ? para desplegar el siguiente cuadro y elegir un registro existente




Modificar o Eliminar

Para modificar se debe digitar el codigo del registro a modificar y presionar el boton modificar, en el caso de eliminar se debe de presionar el boton eliminar
